Active Menu Highlighting Demo

This demonstrates how the active menu highlighting works:

Top Header Menu (when logged in):

  • My Profile
  • Change Password
  • My Tour History
  • Issue Tickets
  • Welcome :
  • Demo User
  • / Logout

Main Navigation Menu:

  • Home
  • Tour Packages
  • Privacy Policy
  • Terms of Use
  • Corporate
  • Contact Us
  • About
  • Enquiry

How it works:

  • Automatic Detection: The system automatically detects the current page using basename($_SERVER['PHP_SELF'], '.php')
  • Active Class: When a menu item matches the current page, it gets the active class
  • Red Background: Items with the active class get a red background (#dc3545)
  • Enhanced Styling: Active items have bold text, rounded corners, and subtle shadows
  • Hover Effects: Active items have enhanced hover effects with slight elevation

Supported Pages:

  • Home: index.php
  • Tour Packages: package-list.php
  • Privacy Policy: page.php?type=privacy
  • Terms of Use: page.php?type=terms
  • Corporate trip: corporate.php
  • Contact Us: page.php?type=contact
  • About: page.php?type=about us
  • Enquiry: enquiry.php
  • Profile: profile.php
  • Change Password: change-password.php
  • Tour History: tour-history.php
  • Issue Tickets: issuetickets.php
